RobotVersion

class RobotVersion

机器人版本号类

机器人版本号类，用于表示机器人版本号

备注

机器人版本号格式为：

  • MMMM 表示次版本号, 范围为0-9999

  • N 表示修订号, 范围为0-9

  • PPP 表示主版本号, 范围为0-9999

  • BIGNUMBER: PPPPMMMMN

  • STRING: PPPPMMMMN Patch 为 0 时，不显示 Patch 部分, 比如 M(4), N(5),P(0) 表示为 45 而非 000000045 (0000,0004,5)

Loop

class Loop

定时循环执行器类

创建一个独立的线程，按照指定周期循环执行用户提供的函数 支持绑定到指定CPU核心以提高性能和确定性

Public Types

using LoopFunction = std::function<void()>

循环函数类型定义

Public Functions

inline Loop(const std::string &name, LoopFunction function, int period_ms, int cpu_core = -1, int sched_priority = 80)

构造函数

参数:

  • name – 循环器名称，用于标识和调试

  • function – 要执行的函数对象

  • period_ms – 执行周期，单位毫秒

  • cpu_core – 要绑定的CPU核心号，-1表示不绑定

  • sched_priority – SCHED_FIFO优先级，-1表示不设置实时调度

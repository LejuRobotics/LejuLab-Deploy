lejusdk-lowlevel 数据类型

SDK 中使用的核心数据结构定义。示例代码中涉及的类型可点击跳转至本页对应说明。

MotorControlMode

enum class leju::MotorControlMode : uint8_t

电机控制模式枚举

定义了机器人电机支持的三种控制模式，用于指定电机的运行方式。

警告

CST 模式下，kp 和 kd 必须设置为 0，否则会叠加 PD 控制律的输出。

Values:

enumerator CST

力矩控制模式，直接发送目标力矩，kp/kd 必须为 0

enumerator CSV

速度控制模式，发送目标速度

enumerator CSP

位置控制模式，发送目标位置、Kp、Kd，内部计算力矩

HardwareState

enum class leju::HardwareState

硬件状态枚举

Values:

enumerator UNKNOWN

未知状态

enumerator STOPPED

已停止

enumerator ERROR

错误状态

enumerator INITIALIZING

初始化中

enumerator READY_OK

就绪，可发送控制指令

HardwareState leju::String2HwState(const std::string &hw_state_str)

RobotCmd

struct RobotCmd

机器人关节控制指令结构体

用于封装机器人关节的控制参数，包括目标位置、速度、力矩以及控制增益等。 支持三种控制模式：CST(力矩控制)、 CSV(速度控制)、CSP(位置控制)。

参见

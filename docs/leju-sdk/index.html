LejuSDK 文档

LejuSDK 是 Leju 机器人软件开发工具包，提供多层次的机器人控制接口。

模块概览

模块

说明

lejusdk-utils

工具库：版本管理、时间工具、定时循环、日志等，SDK 基础依赖

lejusdk-lowlevel

底层控制 SDK，提供电机控制、传感器数据读取等接口（含示例）

lejusdk-highlevel

高层控制 SDK，提供应用层控制接口（含示例） 暂未上线，敬请期待

# leju-vr-control

`leju-vr-control` 是面向使用者的 Quest3 VR 控制节点。它负责将 Quest3 的输入转换为机器人控制行为，包括手臂增量控制、头部控制和速度指令控制。

## 功能概览

当前支持的主要功能：

- 手臂 VR 增量控制
- 头部控制
- 速度指令控制

## 适用机型与限制

| 机型 | 手臂控制 | 头部控制 | 速度控制 | 腰部控制 | 备注 |
| --- | --- | --- | --- | --- | --- |
| `Kuavo4Pro(45/46)` | 支持 | 支持 | 支持 | 不支持 | 无腰部自由度 |
| `Kuavo5(52)` | 支持 | 支持 | 支持 | 不支持 | 暂不提供 VR 腰部控制 |
| `Roban(14)` | 不开放 | 不开放 | 不开放 | 不开放 | 节点启动时直接退出 |

## 启动前检查

建议启动前确认：

- `ROBOT_VERSION` 设置正确
- Quest3 与 PC 处于可通信网络环境
- `leju-remote` 已正常运行
- 当前机器人处于安全环境，适合做 VR 遥操作测试

## 启动方式

```bash
rosrun leju-vr-control quest_vr_control_node
```

通常需要同时运行 `leju-remote`，用于接收 Quest3 数据。

## 操作方式

### 手臂模式切换

- `X+A`
  - 在 `Auto(1)` 和 `External(2)` 之间切换
- `X+B`
  - 在 `KeepPose(0)` 和 `External(2)` 之间切换

三个模式的效果如下：

| 模式 | 值 | 效果 |
| --- | --- | --- |
| `KeepPose` | `0` | 保持当前手臂姿态，适合临时停在当前位置 |
| `Auto` | `1` | 交还给默认自动行为，手臂不再接受当前 VR 增量控制 |
| `External` | `2` | 进入 VR 外部控制模式，按住对应 `grip` 后可进行增量控制 |

### 手臂控制

- 左手 `grip`
  - 控制左臂进入增量控制
- 右手 `grip`
  - 控制右臂进入增量控制
- 双手 `grip`
  - 可同时控制双臂

### 头部控制

头部姿态由头显姿态驱动。

## 按键逻辑

| 操作                         | 功能                                         |
| ---------------------------- | -------------------------------------------- |
| `X+Y`                      | 进入安全停机/关节松弛状态                    |
| `X+A`                      | 在 `Auto(1)` 与 `External(2)` 间切换     |
| `X+B`                      | 在 `KeepPose(0)` 与 `External(2)` 间切换 |
| 左手 `grip`                | 左臂进入增量控制                             |
| 右手 `grip`                | 右臂进入增量控制                             |
| 左摇杆                       | 速度平移                                     |
| 右摇杆                       | 速度旋转                                     |

## 操作流程示例

### 示例 1：手臂控制

1. 启动 `leju-remote`
2. 启动 `leju-vr-control`
3. 按 `X+A` 切到 `External(2)`
4. 按住单侧或双侧 `grip`
5. 移动 Quest3 手柄开始控制手臂

## 限制与常见现象

- 初次按下 `grip` 时，需要实际手臂发生移动后才会进入有效增量跟随
- `Roban(14)` 当前不支持该节点
